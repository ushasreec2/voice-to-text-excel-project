# Voice-to-Text Excel Editor

A Flask-based web application that allows users to update Excel spreadsheets using natural language voice or text commands.

## Features

* Upload Excel files
* Insert values into cells
* Generate Excel formulas using commands
* Preview spreadsheet updates
* Download updated files

## Technologies

* Python
* Flask
* OpenPyXL
* HTML
* JavaScript

## Run Locally

```bash
pip install flask flask-cors openpyxl
python app.py
```
